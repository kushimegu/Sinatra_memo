# Sinatra_memo
## アプリの概要
Sinatraを使ったJSON形式でデータを保存する簡単なメモアプリです。メモ一覧の表示、新規メモの作成、メモ詳細の表示と編集、削除を行えます。
## アプリの立ち上げ方
1. リポジトリのクローニング

```bash
$ git clone https://github.com/kushimegu/Sinatra_memo.git
```

1. リポジトリへの移動

```bash
$ cd Sinatra_memo
```

1. Gemのインストール

```bash
$ bundle install
```

1. アプリの実行

```bash
$ ruby memo.rb
```

1. URLへのアクセス
ブラウザでlocalhost:4567/memosにアクセスする
